# GitHub Action: JupyterLite Xeus build

## Usage

- Config: see [`environment.yml`](https://github.com/casperdcl/jupyterlite-xeus-pages/blob/v1/environment.yml)
- CI: see [`.github/workflows/web.yml`](https://github.com/casperdcl/jupyterlite-xeus-pages/blob/v1/.github/workflows/web.yml)
- Arguments: see [`action.yml`](https://github.com/casperdcl/jupyterlite-xeus-pages/blob/v1/action.yml)

## Alternatives

Similar to [`yuvipanda/repo2jupyterlite-action`](https://github.com/yuvipanda/repo2jupyterlite-action) but more up-to-date and much lighter-weight (fully self-contained in one composite action with no external dependencies).
